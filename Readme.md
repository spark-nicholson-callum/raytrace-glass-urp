# レイトレーシング・ガラスエフェクトシェーダー

Unity 6.4 (URP) 環境向けのガラスエフェクトシェーダーです。ハードウェアレイトレーシングを利用して屈折と反射を計算し、物理的に正確な描画を実現しています。
パフォーマンスを考慮し、スクリーンスペースと正確なレイトレーシングを組み合わせたハイブリッドアプローチを採用しました。
